Methodology

  • Returns use Rebated_ROI per trade (decimal). Graph starts at 0% and adds each trade's ROI without compounding.

  • Sharpe/Sortino ratios annualize per-trade stats using the same time-weighted frequency. Max drawdown follows the linear equity curve.
